## Setup
1. **Install dependencies**
   `ash
   cd backend
   npm install
   `
2. **Create a .env file** based on the .env.example.
3. **Run the server**
   `ash
   npm run dev   # development with nodemon
   # or
   npm start     # production
   `
   The server will start on the port defined in .env (default 5000).
